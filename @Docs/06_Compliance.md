# CHAPTER 6 — COMPLIANCE ENGINE

**6.1 — PURPOSE**
Transforms documents (certificates) into structured rules and tasks.

**6.2 — PIPELINE**
Upload → Extractor (AI) → Critic (Validation) → Review (Human) → Activation.

**6.3 — RULES**
*   **Renewal:** Generates bookend tasks (Insurance).
*   **Inspection:** Periodic tasks (Alarm tests).

**6.6 — EXPIRY**
Status: Valid → Expiring (30 days) → Expired.
